# User Stories — forlove10grams

## Creator（作者本人）

### 記錄

- 作為 creator，我想建立一本新書，讓我可以記錄一段特定的經歷
- 作為 creator，我想在手機上快速開始一筆記錄，不需要經過複雜的建立流程
- 作為 creator，我想在一頁裡放多張照片（carousel），讓我可以用影像敘述一段故事
- 作為 creator，我想在一頁裡放一支影片加上文字說明，讓我可以記錄動態的片段（例如潛水、料理過程）；上傳後系統自動轉檔為 HLS，轉檔完成前我看到「轉檔中」提示，完成後直接播放
- 作為 creator，我想替書設定封面圖和標題，讓書看起來有辨識度
- 作為 creator，我想在頁面內容中使用 `${Nickname}` 和 `${MyNickname}` 佔位符，讓每位讀者在閱讀時看到屬於自己的稱呼

### 管理與瀏覽

- 作為 creator，我想在 dashboard 看到我所有的書，讓我可以快速找到要繼續編輯的那本
- 作為 creator，我想在 dashboard 依排序（新→舊、舊→新、標題）和發布狀態（全部、已分享、草稿）篩選我的書，讓書多了之後也能快速定位
- 作為 creator，我想搜尋我的書（by 標題或標籤），讓我在書多了之後還能快速找到「那次潛水」是哪本
- 作為 creator，我想為書本加上自由文字標籤，讓我可以分類書本（例如「潛水」「料理」「散步」）
- 作為 creator，我想在 dashboard 直接開啟標籤管理視窗，新增或刪除書本的標籤，不需要進入編輯頁
- 作為 creator，我想調整書內頁面的順序，讓敘事更流暢

### 協作與分享

- 作為 creator，我想邀請一位朋友成為 editor，讓她可以繼續寫入她的視角；邀請時我必須填寫一封交接信，讓她在閱讀書末時感受到被交付的儀式感
- 作為 creator，我想在 dashboard 看到書目前的 editor 是誰，必要時可以移除 editor，讓書回到只有我能編輯的狀態
- 作為 creator，我想產生一個分享連結，把書傳給知道這段故事的朋友
- 作為 creator，我想查看目前的分享連結狀態（URL、到期時間），並在必要時撤銷它
- 作為 creator，我想延長分享連結的有效期，而不需要更換 URL
- 作為 creator，我想知道透過分享連結讀了這本書的人有哪些，並可以移除特定讀者
- 作為 creator，我想在 dashboard 看到書被讀者 like 了幾次，但不需要知道是誰
- 作為 creator，即使邀請了 editor，我仍然要能繼續編輯這本書，不會失去存取權
- 作為 creator，我知道我邀請的 editor 也能產生分享連結，由她決定要不要把書傳給更多人

## Editor（受邀的協作者）

### 登入後的落腳點

- 作為 editor，我登入後想在 dashboard 看到「我可以編輯的書」，讓我不需要每次都靠連結才能找到
- 作為 editor，我想搜尋或篩選我有編輯權的書（by 標題或標籤）

### 寫入與閱讀

- 作為 editor，我想在受邀的書裡新增頁面，補充我自己的記憶和心得
- 作為 editor，我想上傳我拍的照片和影片，融入那本書的記錄
- 作為 editor，我想為受邀的書新增或刪除標籤，讓我也能參與書的分類管理
- 作為 editor，我在書末閱讀完所有頁面後，可以看到 creator 寫給我的交接信，並從那裡直接進入編輯頁

### 分享與管理

- 作為 editor，我想產生分享連結，把這本書傳給其他知道這段故事的人
- 作為 editor，我想查看目前的分享連結狀態，並在必要時撤銷或延長它
- 作為 editor，我想知道透過分享連結讀了這本書的人有哪些，並可以移除特定讀者
- 作為 editor，我理解我無法移除 creator 的存取權

---

## Reader（透過分享連結進入的讀者）

### 閱讀

- 作為 reader，我點開分享連結後，如果還沒登入，我期待被引導完成登入，然後自動回到那個連結
- 作為 reader，我透過分享連結第一次進入書本後，系統會記錄我的閱讀身份，讓我之後可以直接從 dashboard 再找到這本書
- 作為 reader，我想順暢地瀏覽一本書的所有頁面
- 作為 reader，我想知道我讀到哪裡了（閱讀進度）
- 作為 reader，我在首次登入後可以設定我的顯示名稱（也可以跳過），這個名字會在閱讀時融入作者寫給我的內容
- 作為 reader，我希望在捲動時能流暢地載入更多頁面，不用等全部頁面一次載完
- 作為 reader，我在桌機閱讀時，TOC 會即時標示我目前讀到哪一頁
- 作為 reader，我想在讀完一本書後 Like 它，讓作者知道我喜歡

### 找回

- 作為 reader，我登入後想在 dashboard 看到「分享給我的書」，讓我之後想再看的時候不用找連結
- 作為 reader，我想搜尋我讀過的書
